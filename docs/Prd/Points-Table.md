## Sorting Quality Multiplier Summary

| Level | Multiplier | Requirements |
|-------|------------|--------------|
| **Gold** | 1.5x | Perfectly sorted, clean, prepared (labels removed, rinsed, crushed, caps separate) |
| **Silver** | 1.3x | Well sorted, mostly clean (labels on but rinsed, not crushed) |
| **Bronze** | 1.1x | Sorted but needs work (labels on, some residue) |
| **Mixed** | 1.0x | Combined recyclables in one bag |
| **Contaminated** | 0.5x | Wrong items mixed in (food waste with paper) |
| **Hazardous** | 0x + Warning | Dangerous items in wrong bin (batteries in plastic) |

## Cleanliness Factor

| Level | Factor | Description |
|-------|--------|-------------|
| **Clean** | 1.2x | Rinsed, no residue, dry |
| **Average** | 1.0x | Some residue, mostly clean |
| **Dirty** | 0.7x | Food residue, wet, smelly |
